# 🕹️ 8-Bit Retro Portfolio

> A stylized, pixel-art inspired personal portfolio built with **Nuxt 4** and **NES.css**.

Welcome to my digital playground. This portfolio is designed to look and feel like a classic 8-bit RPG, featuring a responsive dashboard, localized content, and a clean, organized skills section.

## 🚀 Live Demo
Visit the live site: [portfolio.francescotesta.net](https://portfolio.francescotesta.net)

---

## 🏃 Local Development

### Installation
```bash
npm install
```

### Dev Server
```bash
npm run dev
```

### Build & Generate static site
```bash
npm run generate
```
